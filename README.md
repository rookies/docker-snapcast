docker-snapcast
===============

Docker images of [snapcast](https://github.com/snapcast/snapcast). Built for amd64 and arm64.

## snapserver
Includes [snapweb](https://github.com/snapcast/snapweb) and
[librespot](https://github.com/librespot-org/librespot).

```
docker run --rm ghcr.io/rookies/snapserver:latest --help
```

## snapclient
```
docker run --rm ghcr.io/rookies/snapclient:latest --help
```
